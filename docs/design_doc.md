# Design Doc

## 项目定位

一款融合三国武将对战、麻将胡牌结构、Roguelike 构筑和高动效爆发反馈的单机游戏。

长期开发总纲以 `docs/godot_development_framework.md` 为准。

## 当前 1.0 基础战斗闭环

- 使用 136 张无花麻将牌。
- 起手摸 16 张，玩家选择 13 张作为初始手牌，剩余 3 张弃置。
- 每回合抓 3 张，必须打出 3 张。
- 打出的 3 张按优先级判定：刻子、顺子、对子 + 单张、散牌。
- 顺子和刻子进入明组区。
- 当前手牌中的顺子和刻子作为暗组候选。
- 常规胡牌结构为 `AAA + BBB + CCC + DDD + EE`。
- 胡牌大招按明组、暗组、将牌、散打牌的权重重算收益，并乘以胡牌倍率。
- 敌人按 3 回合循环行动：准备、轻量干扰、主要攻击。

## 1.0 不做

- 花牌、季牌。
- 七对、十三幺和其他特殊胡法。
- 吃碰杠、副露、立直、宝牌。
- 完整武将技能、武器印记、商店、地图和元进度。
