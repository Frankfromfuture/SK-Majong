# Design Doc

## 项目定位

一款融合改良麻将牌型机制、Roguelike 构筑和三国题材的单机游戏。

## 当前约束

- 麻将规则采用「巡目得分流」，不是传统日麻。
- MVP 不做字牌、立直、宝牌、吃碰杠、兵法、存档和元进度。
- 武将名字在 MVP 中可用真名占位。

## Phase 1 规则澄清

- 策划文档同时出现「每巡选 1-7 张」和「一气通贯 9 张」；核心层暂支持 1-9 张，UI 层限制待 Phase 3 前确认。
- 策划文档中「役满 = 7 张完整结构：雀头 + 面子 + 面子」存在张数矛盾。Phase 1 暂定为 7 张结构：雀头 + 1 个三张面子 + 1 个两张搭子或对子。
