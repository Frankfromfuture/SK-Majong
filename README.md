# Sangoku Mahjong

三国题材武将对战麻将 Roguelike 原型。

## 当前玩法

- 标准 136 张麻将牌。
- 起手 13 张，每次行动开始按上次出牌数量补牌，手牌超过 14 张时必须先弃牌。
- 每次行动可打出 0-3 张；4 张完全相同可作为杠打出。
- 打出的顺子、刻子、杠会进入战场招式区，单张、散牌、对子只做即时结算。
- 条是近战攻击，优先攻击敌方军防；饼是远程攻击，敌方军防为 0 后攻击城防；万只增加我方军队防御。
- 敌人与我方都采用双层生命：敌方军防/城防，我方军队防御/将领血条。
- 常规胡牌触发“全军总攻”，不会直接结束战斗；若敌方城防未归零，则清空手牌和招式区后重新抓 13 张继续战斗。
- 敌方每 3 次玩家行动攻击一次，优先扣我方军队防御，溢出伤害扣将领血条。
- 字牌功能仅做 2.0 预留，v1.0 无基础点数和特殊效果。

## 运行

安装 Godot 4.3+ 后，打开 `project.godot`，运行主场景。

## 目录

- `src/core/`：基础规则逻辑。
- `src/scenes/`：主菜单、战斗、规则、结算场景。
- `src/ui/`：当前游戏使用的 UI 和 3D 麻将牌显示。
- `assets/`：当前游戏加载的图片资源。
